# Waterfall Model - Smart AI Fridge Development

## Course Name
CT&DT - Galgotias University

## Batch Number
Batch No: 1

## Task Details
Task M2(T3) - Waterfall Models and Product Development

## Project Description
This project demonstrates the Waterfall Model applied to the development of a Smart AI Fridge. The model follows a sequential approach starting from requirement gathering to maintenance.

## Waterfall Stages
1. Requirement Gathering  
2. System Design  
3. Implementation  
4. Testing  
5. Deployment  
6. Maintenance
